nidara-repo

A pacman binary repository for Nidara — pre-built copies of the Astal libraries, the ags CLI, and appmenu-glib-translator.

Import the signing key (nidara.gpg):

curl -fsSL https://nidara-project.github.io/nidara-repo/nidara.gpg | sudo pacman-key --add -
sudo pacman-key --lsign-key 80B0AC8C36A43611A8619959B06B716279F755A9

Add to the end of /etc/pacman.conf:

[nidara]
SigLevel = Required DatabaseOptional
Server = https://nidara-project.github.io/nidara-repo/$arch

Then sudo pacman -Sy && sudo pacman -S aylurs-gtk-shell.

Packages live under x86_64/. All packages and the repo database are signed; the key is committed in the source repo.
